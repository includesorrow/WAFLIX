## 소개
WAFLIX란 WATCHA와 NETFLIX를 합한 영화 스트리밍 사이트입니다.

사이트 내의 기능으로는 WATCHA, NETFLIX같이 사이트내에 추천 알고리즘을 구현하여 고객에게 자신의 성향에 맞는 추천영화들을 제공해주며,  관리자 페이지 내에서 판권분석, 수익률 분석 알고리즘을 구현하여 관리자페이지에서 확인이 가능합니다.
